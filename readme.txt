CronosForm

CronosForm es una plugins que pretende minimizar el tiempo de población  y registro de un determinado formulario web de forma. CronosForm viene en dos formas diferentes. Usted puede usarlo de forma local o remota.
Adicionalmente se esta extendiendo la funcionabilidad para que soporte la población y registro con los componentes del Framework Materializecss para una mejor experiencia de las páginas que prentendas desarrollar.
